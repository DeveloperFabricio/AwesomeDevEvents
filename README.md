## AwesomeDevEvents - Criando REST APIs com ASP.NET Core


### Tecnologias e ferramentas utilizadas

 - Visual Studio 2022
 - ASP.NET Core 7
 - EF Core
 - Swagger
 - AutoMapper
 - SQL Server
   
### Funcionalidades

 - Cadastro, Listagem, Detalhes, Atualização, e Remoção de Evento
 - Cadastro de palestrantes
